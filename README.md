# carhackingwiki

This is the repository for the configuration of Car Hacking Wiki, a mediawiki-based wiki family. It's intended to be a family of wikis focusing on the technical internals of vehicles, such as components and subsystems, electrical, electronic and data specifications, information on known modifications and projects, etc.

At this stage it is comprised only of the [Nissan Leaf Wiki](https://nissanleaf.carhackingwiki.com/index.php/Nissan_Leaf) which focuses on the Nissan Leaf.

If you want to make a different one under the same family, raise an issue or PR.
